# 🧅 Rakkyo (ラッキョ)

> **「間違えても大丈夫！ラッキョくんといっしょに、あきらめない心を育てよう 🧅」**

Rakkyo は、中学生が自発的に・あきらめずに学習を継続できるように設計された、**AI チューター連動型次世代学習プラットフォーム**です。
学習をキャラクター「ラッキョくん」が優しくナビゲートし、単なる正誤判定にとどまらず、**「3段階の段階的ヒント」**や**「保護者との温かい応援メッセージ連携」**を通じて、生徒のレジリエンス（Grit: あきらめない心）を育みます。

---

## 🛠️ 技術スタック & アーキテクチャ

本プロジェクトは **pnpm ワークスペース** を採用したモノレポ構成で構築されています。各パッケージの依存関係やビルドは **Turbo** によって高速に管理・キャッシュされます。

### モノレポ構造

```text
├── apps/
│   ├── web/        # Next.js (TypeScript) / Tailwind CSS を採用した Web フロントエンド
│   ├── api/        # Express (TypeScript) / Prisma を採用したバックエンド API サーバー
│   └── mobile/     # モバイル向け React Native / Expo アプリケーション
│
├── packages/
│   ├── ai-tutor/   # AI プロンプトエンジニアリング・対話ロジックのコアモジュール
│   ├── curriculum/ # 数学など各教科の単元・レッスン・問題データの静的定義
│   ├── shared/     # フロント・バックで共有する TypeScript の共通型定義・ユーティリティ
│   └── ui/         # モノレポ共通のUIデザインシステム・コンポーネント
│
├── docs/           # 将来のマルチテナント設計書などのアーキテクチャドキュメント
└── README.md       # 本ドキュメント
```

## ✨ 主要な独自フィーチャー (特色)

### 1. 3段階AIヒントシステム (Progressive Hint System)
生徒がわからない問題に直面した際、すぐに答えを教えるのではなく、段階的に理解を促すヒントをAIが生成します。
* **段階 1 (言い換え)**: 問題の意図や言葉の意味をかみ砕いて説明します。
* **段階 2 (イメージ図)**: 視覚的に整理し、数式や文章の構造をビジュアル化して伝えます。
* **段階 3 (手順ガイド)**: 解くための具体的なステップ（次に何をすべきか）の道筋を示します。

### 2. AIコスト爆発ガード & 優しいフォールバック (Cost Control)
教育現場での安定した運用コスト管理のため、1日のAIヒント利用上限を設定しています。
* **1日20回の上限**: 1人の生徒につき1日あたり20回（`MAX_AI_HINTS_PER_DAY` で調整可能）にAIヒント出力を制限。
* **応援付き静的ヒント**: 上限に達した場合は、自動的に問題固有の「静的ヒント」へスムーズにフォールバック。ラッキョくんからの前向きな応援メッセージ（**「今日はたくさんラッキョくんとお勉強したね！代わりに問題のヒントをあげるよ 🧅」**）を添えて表示するため、生徒の学習意欲を損ないません。

### 3. 不適切入力対策(Abuse Filter) & 連投制限(Spam Control)
安心・安全な学習環境づくりのため、AI対話へのセキュリティガードを徹底しています。
* **Abuse フィルター**: 罵倒語・不適切表現・お勉強に関係のない対話（「遊ぼう」など）を検知。検知時はAI出力をバイパスし、ラッキョくんの愛嬌ある警告定型文（**「ラッキョくんとはお勉強のお話をしてほしいな！🧅」**）を返却。
* **3秒の連投ブロック**: AIへの過度な連打リクエストを一時的に制限し、サーバー負荷とコストの急増を防ぎます。

### 4. 保護者用ダッシュボード (Parent Dashboard)
保護者専用ページ (`/parent`) では、お子様の学習成果をただ点数で評価するのではなく、**「プロセス」や「あきらめない努力」**を可視化します。
* **Grit (あきらめない心) スコア**: ヒントを読んで自力で正解までたどり着いた割合を算出。
* **可観測性の高い美しいグラフ**: SVGを用いた日別アクティビティグラフ、単元の学習進捗バー、3段階ヒントの活用比率を表すドーナツチャートなどを搭載。
* **応援メッセージ機能 (メッセージ連携)**: 保護者からお子様へ150文字以内の応援メッセージを送信可能。メッセージはお子様の学習画面に届き、既読状態はリアルタイムで保護者画面へフィードバックされます。

> [!NOTE]
> **【設計の美しさ：関心の分離】**
> 保護者用ダッシュボードのバックエンド集計ロジックは `parentStatsService.ts` へ完全カプセル化され、フロントエンドUIも4つの単一責務コンポーネント（`DashboardTab`, `WeaknessTab`, `WeeklyTab`, `MessageTab`）に美しく分割リファクタリングされており、極めて高い保守性を備えています。

### 5. 法務安全同意フロー (Parental Consent)
中学生や子どもたちが安全に本サービスを開始できるよう、登録時に「保護者の同意」を必須チェック化しています。
* 子どもたちにとっても読みやすく分かりやすい、平易で温かみのある言葉で書かれた「利用規約」と「プライバシーポリシー」ページを完備。

### 6. コラボレーティブ・ラーニングとピア応援 (Collaborative Learning)
生徒同士が競い合うのではなく、お互いの「あきらめない姿勢（Grit）」を称え合うことで、孤独感を解消し、優しく連帯できる「共創・共習」空間を構築しました。
* **バーチャル学習室「ラッキョ・ルーム」**: クラスメイトたちの非同期プレゼンスをふわりと浮かぶアバターアニメーションと吹き出しで表示。友達をタップして「あきらめないね！🧅」「Grit!💪」といった応援スタンプを非同期に送信可能。
* **液体のように揺れる「クラス共習ミッション」**: 個人の正解数ではなく、クラス全員で合計学習時間の目標突破を目指す協力型ミッション。進捗をリッチなSVG液体ウェーブアニメーションで表現。
* **匿名つまずき共有ボード「みんなのひらめき💡」**: 「この問題は難しかったけど、こう考えたら解けた！」をつまずき克服付箋として匿名（野菜ニックネーム自動割り当て）で共有。投稿には AI 安全フィルターを適用し、不適切な表現があればラッキョくんが優しく対話的に警告ブロック。
* **親子お祝いフィードバックループ**: 生徒が難問をやり抜いてクリアした際に、保護者にワンクリックで「がんばり応援レポート（AI精密診断付き）」への一時お祝いリンクを生成・シェア。保護者からスタンプやメッセージが返送されると、生徒ダッシュボードへお祝い通知が届き、ラッキョくんボイス（TTS）で感情を込めて読み上げ。

## 🚀 開発環境のセットアップと起動方法

### 前提条件
* **Node.js**: `v18` 以上推奨
* **pnpm**: `v8` または `v9`
* **Docker / PostgreSQL** (本番またはローカル Prisma 同期用)
  * ※ローカル開発環境では、DB接続エラー時に自動的にメモリ上の **Mock DB** にシームレスにフォールバックする「優しきフォールバック設計」が組み込まれているため、DBなしでも手軽に起動して動作を確認できます。

### 1. 依存関係のインストール
プロジェクトのルートディレクトリで以下を実行します。

```bash
pnpm install
```

### 2. ローカル開発サーバーの起動
フロントエンド（Next.js）およびバックエンド（Express）を同時に立ち上げます。

```bash
pnpm run dev
```
* **フロントエンドWeb**: [http://localhost:3000](http://localhost:3000)
* **バックエンドAPI**: [http://localhost:4000](http://localhost:4000)

### 3. テストの実行
モノレポ内のすべての統合テストおよびユニットテストスイートを実行します。

```bash
pnpm test
```

### 4. プロダクションビルド
TypeScriptの厳密な型チェックと最適化ビルドを走らせます。

```bash
pnpm build
```
